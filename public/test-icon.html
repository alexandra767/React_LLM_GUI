Instructions:

  1. Save your brain-lightning image as brain-lightning.png in the public folder
  2. Run: node convert-image-to-base64.js public/brain-lightning.png
  3. Copy the base64 string from brain-lightning-base64.txt
  4. Update src/components/Chat/BrainLightningIcon.js with the base64 string
  5. Refresh this page to see how the icon will look
