🔑 Google Drive API Setup

Step 1: Get Google Cloud Credentials

1. Go to Google Cloud Console

2. Create a new project or select existing

3. Enable Google Drive API

4. Create OAuth 2.0 Client ID credentials

5. Add these authorized redirect URIs:

http://localhost:3000 https://developers.google.com/oauthplayground

Step 2: Enter Your Credentials

Step 3: Get Access Tokens

After saving credentials, you'll need to get access and refresh tokens:

1. Go to OAuth 2.0 Playground

2. Click the gear icon (top right) and enter your Client ID and Secret

3. In Step 1, select these scopes:

https://www.googleapis.com/auth/drive.readonly https://www.googleapis.com/auth/drive.file

4. Authorize APIs → Exchange authorization code for tokens
